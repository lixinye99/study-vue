## 对vuex使用的总结
1.state中的数据要进行修改，不能直接修改，必须通过mutations
2.如果组件想要从state中获取数据需要this.$store.state.xxx
3.如果组件想要修改数据必须使用mutations提供的方法，需要通过this.$store.commit('方法名'，唯一的一个参数)
4.如果state中的数据，在对外提供的时候需要做一层包装，推荐使用getters
